How to Install

ESign is a free tool and very easy to install. This guide will show you how to install it. Everything in this is completely safe.
  1. Download the DNS files from the button below
  2. Make sure the DNS is enabled
  3. Download the Certificates from the button below
  4. Download each of the ESign Apps until you find one that works (Each app is different per phone)
